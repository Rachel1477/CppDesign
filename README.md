# C++课程设计-餐厅后台系统
项目的技术栈：Qt Creator 4.11.1 ，SQLite3 ，qmake  

# 注意事项
## 1
本项目用Qt Creator 4.11.1 制作，运行时请确保你的QT版本高于它

## 2 
项目涉及本地嵌入式数据库SQLite3，所有的数据都存储在mydatebase.db中（当前目录下），如果想要查看这个.db文件的内容，请自行下载[数据库可视化工具](https://sqliteexpert.com/)
